- **Day-detail "Remove period"** — `_PeriodMarkAction` now renders a red "Remove period" button (with confirm dialog) whenever the tapped day overlaps any logged period; "Mark period end" still shows for in-progress periods; "Mark period start" only when no overlap and no in-progress one exists.
- **Settings → Cycle section** — new `_CycleSettingsSection` between Location and Triggers: "Log/End period" button mirroring the Today screen, plus a list of recent periods with per-row delete (same confirm dialog).
